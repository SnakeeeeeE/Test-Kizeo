Arborescence

Le projet contient 5 fichiers:

  • index.html => le fichier actuel
  • scripts.js => contient le javascript du projet
  • style.css => contient le style CSS du projet
  • modal.js => contient le javascript permettant d'afficher un modal
  • modal.css => contient le CSS permettant d'afficher un modal

Fonctionnement

Tout d'abord il faut s'identifier en entrant son identifiant/mot de passe et le code de l'entrepise. En appuyant sur le bouton "transmettre" on va récupérer un token qui nous permettra ensuite d'afficher les formulaires

Travail effectué

J'ai essayé de répondre à l'ensemble des demandes spécifiées dans le sujet, à savoir:

  • Connexion et récupération du token en AJAX via le bouton "transmettre". Le token est stocké à la fois dans une variable et aussi dans un cookie
  • Affichage des différents formulaires à l'aide du bouton "afficher forms" après s'être identifié
    • La liste des formulaires apparaît dans un tableau html, en entête se trouve les informations principales (ID, Nom, Class, date)
    • La date apparaît au format FR long
    • Les données peuvent être triées (ASC/DESC) à l'aide d'un simple clic sur les différents labels de l'entête du tableau
    • Lors d'un clic sur une ligne du tableau, affiche les informations du formulaire associé en ouvrant un modal
